Thông Tin Sản Phẩm

Gamepad Không dây Bluetooth PS/4 Black
1.450.000đ
Mô Tả Sản Phẩm

- Gamepad Không dây Bluetooth cho máy tính - điện thoại - máy game Console PS/4
- Trọn Bộ bao gồm Gamepad + Dây sạc USB
- Thiết bị hỗ trợ : máy tính - Laptop - điện thoại - Máy tính bảng - máy game Console PS/4
- Hệ điều Hành Hỗ trợ : Android + Window + iphone , ipad iOS13 trở lên
- Kết nối : Bluetooth hoặc cắm dây USB trực tiếp
- hỗ trợ đầy đủ tính năng ( cảm ứng + loa + rung + led )
- Pin Sạc 600 mah
- Sản phẩm cho cảm giác thoải mái và hỗ trợ nhiều game trên window

Hướng Dẫn Kết Nối Bluetooth (Yêu cầu thiết bị phải có bluetooth , trên PC tải về phần mềm " DS4Windows.zip " giải nén và cài driver) :
1/ Trên tay nhấn giữ trước " SHARE " ---> sau đó nhấn giữ thêm " PS " đến khi ( led nháy nhịp đôi ) thì thả ra
2/ Mở bluetooth trên thiết bị , dò và kết nối
3/ Ở lần kết nối thứ 2 trở đi chỉ cần nhấn " PS " ( led nháy đều ) , không cần làm lại bước " 1 "
4/ Để ngắt kết nối tay cầm với thiết bị : tắt bluetooth trên thiết bị hoặc nhấn giữ " PS " trên tay game
5/ Để test Tay game : - Trên máy tính vào control panel --> devices and printer --> chuột phải lên icon gamepad --> game controler setting --> properties --> Test - Trên điện thoại tải app PS4 Controller test
